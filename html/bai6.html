HTML (the Hypertext Markup Language)
and CSS (Cascading Style Sheets) are
two of the core technologies for building
Web pages. HTML provides the structure of
the page, CSS the (visual and aural) layout,
for a variety of devices. Along with graphics
and scripting, HTML and CSS are the basis.
